#吃豆人

玩法介绍：
===
基础玩法：在有限的时间内控制人去触碰豆子

难度分为简单（难度1）：碰壁死亡、中等（难度2）：固定仙人掌、困难（难度3）：随机仙人掌

积分消耗计算公式：难度等级*(时间限制/10)-1(积分超过消耗的10倍则无法进入)

每次碰到豆子加的分值为难度等级（若时间没到则只会给十分之一）

应用特色：
===
1.摇杆移动
